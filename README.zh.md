## 概览

`myprofile` 是 Eric Wang 的个人主页网站 —— 一个多页面 React 应用，集成了项目展示、技术文章、交互式 ML 实验室和 AI 驱动的知识库。既是专业履历展示，也是个人持续工作的动态记录。

---

## 页面

| 页面 | 描述 |
|------|------|
| `Home` | 首页：Hero 区域、技能展示与精选内容 |
| `Portfolio` | 项目作品集，附详细案例分析 |
| `Articles` | 技术文章与学习笔记 |
| `About` | 个人背景、教育经历与联系方式 |
| `KnowledgeBase` | AI 辅助的知识管理与搜索 |
| `MLLab` | 交互式机器学习实验与可视化 |

---

## 快速开始

**前置要求：** Node.js

```bash
# 安装依赖
npm install


# 启动开发服务器
npm run dev
```
