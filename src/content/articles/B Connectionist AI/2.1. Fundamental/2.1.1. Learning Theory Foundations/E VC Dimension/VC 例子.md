# 三、四个点的情况（不能打散）

现在我们取四个点：

A = (0,0)  
B = (1,0)  
C = (1,1)  
D = (0,1)

这是一个正方形。

现在我们这样标记：

A = 1  
C = 1  
B = 0  
D = 0

也就是：

对角线两个为1，  
另外两个为0。

这就是经典 XOR 模式。

问题：

有没有一条直线可以把 A,C 和 B,D 分开？

答案是：没有。

因为：

- A 和 C 在对角线两端
    
- B 和 D 在另外对角线
    

任何直线都会：

要么把一个 1 和一个 0 分在同一边，  
要么无法同时分开对角。

这就是线性不可分。

所以：

4个点不能被“任意标记”后全部可分。

因此：

二维直线分类器的 VC dimension = 3

---

# 四、直觉解释

直线是一个“一刀切”的模型。

它只能：

- 切出一个半平面
    

而三角形最多只需要一次切割就能实现任意分法。

但四个点的某些标法需要“两次切割”。

直线做不到。
