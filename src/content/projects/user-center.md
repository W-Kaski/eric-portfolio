A robust, enterprise-grade User Management Center built with a modern React frontend and a Spring Boot backend. 

## Key Features
- **Secure Authentication:** Implemented robust session-based authentication with comprehensive server-side input validation and custom encryption for user credentials.
- **Admin Dashboard:** Built an intuitive dashboard using Ant Design Pro for administrators to manage users (CRUD operations).
- **Automated Code Generation:** Leveraged MyBatisX to automatically generate domain, mapper, and service layers from database schemas, significantly increasing development speed.
- **RESTful API Design:** Strict adherence to REST principles using Spring MVC, with custom exception handling and standardized response payloads.

## Architecture & Deployment
- Orchestrated frontend and backend integration via Nginx reverse proxies to solve CORS and routing issues.
- Designed database schemas in MySQL with optimal indexing strategies for fast user querying.
